## 🚀 Overview

terminal-npm-intellisense helps you complete npm, pnpm, yarn, and bun commands inside the VS Code terminal. It is built for Windows users who want faster command entry and fewer typing mistakes.

If you often run package scripts or switch between Node tools, this app gives you command hints as you type. It is meant to make terminal work easier in VS Code.

## ✨ What It Does

- Shows command autocomplete in the VS Code terminal
- Helps with package manager commands
- Supports npm, pnpm, yarn, and bun
- Helps with script names from your project
- Reduces spelling mistakes in long commands
- Speeds up common terminal work

## 🔧 How It Helps

When you type a command in the terminal, the app can suggest the next part. For example, it can help when you start typing:

- `npm run`
- `pnpm add`
- `yarn`
- `bun run`

This is useful when you do not remember the full command or when you want to move faster. It can also help with script names in projects that have many commands.

## 🛠️ Troubleshooting

### The app does not open
- Check that the download finished
- Run the file again
- Make sure Windows did not block the file
- Try downloading it again from the releases page

### No suggestions appear
- Open a project folder in VS Code
- Make sure the terminal is active
- Check that your project has the package manager files it needs
- Try typing the first few letters of the command

### Script names do not show
- Make sure the project has a `package.json` file
- Check that the file includes scripts
- Reopen the project folder in VS Code
- Restart VS Code if needed

### The terminal feels slow
- Close extra terminal tabs
- Restart VS Code
- Open a smaller project folder
- Check that no other extensions are causing conflicts
